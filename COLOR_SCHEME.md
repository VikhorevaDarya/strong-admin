# Цветовая схема Strong Admin - Оранжевый и Черный 🟠⚫

## ✅ Реализовано

Интерфейс админки полностью адаптирован под цвета вашего логотипа - оранжевый и черный.

## 🎨 Цветовая палитра

### Основные цвета

```css
/* Оранжевый (Primary) */
--primary-main: #FF6B00     /* Яркий оранжевый */
--primary-light: #FF8C33    /* Светлый оранжевый */
--primary-dark: #E55D00     /* Темный оранжевый */

/* Черный (Secondary) */
--secondary-main: #1a1a1a   /* Черный */
--secondary-light: #2d2d2d  /* Темно-серый */
--secondary-dark: #000000   /* Чистый черный */

/* Фон */
--background-default: #f8f8f8  /* Очень светлый серый */
--background-paper: #ffffff    /* Белый */

/* Текст */
--text-primary: #1a1a1a    /* Черный */
--text-secondary: #666666  /* Серый */
```

### Дополнительные цвета

```css
--warning: #FF6B00   /* Оранжевый */
--info: #FF8C33      /* Светлый оранжевый */
--error: #d32f2f     /* Красный */
--success: #2e7d32   /* Зеленый */
```

## 🎯 Где используются цвета

### 1. AppBar (Верхняя панель)
- **Фон:** Черный (#1a1a1a)
- **Логотип:** Оранжевый (из PNG)
- **Текст:** Белый (#ffffff)

```
┌────────────────────────────────────┐
│ ⚫ [LOGO] Strong Admin    👤       │ ← Черный фон
└────────────────────────────────────┘
```

### 2. Боковое меню (Sidebar)
- **Фон:** Белый (#ffffff)
- **Активный пункт:** Оранжевый фон (rgba(255, 107, 0, 0.15))
- **Активная граница:** Оранжевая (#FF6B00)
- **Активный текст:** Оранжевый (#FF6B00)
- **Hover:** Светлый оранжевый (rgba(255, 107, 0, 0.08))

```
┌─────────────────┐
│ 📦 Товары       │ ← Обычный
│ 🟠 Склады       │ ← Активный (оранжевый)
│                 │
└─────────────────┘
```

### 3. Кнопки
- **Primary кнопки:** Оранжевый градиент
  - Градиент: `linear-gradient(135deg, #FF6B00 0%, #FF8C33 100%)`
  - Hover: `linear-gradient(135deg, #E55D00 0%, #FF6B00 100%)`
  - Shadow при hover: Оранжевое свечение

```css
[Создать] ← Оранжевый градиент
[Сохранить] ← Оранжевый градиент
```

### 4. Таблицы
- **Заголовки:** Светло-серый фон (#fafafa)
- **Нижняя граница заголовков:** Оранжевая (#FF6B00, 2px)
- **Hover строки:** Светлый оранжевый (rgba(255, 107, 0, 0.04))

```
┌─────────────────────────────────┐
│ ID │ Название │ Цена          │ ← Оранжевая граница
├────┼──────────┼───────────────┤
│ 1  │ Товар 1  │ 1000₽        │
│ 2  │ Товар 2  │ 2000₽        │ ← Hover = светлый оранжевый
└─────────────────────────────────┘
```

### 5. Карточки
- **Фон:** Белый (#ffffff)
- **Граница:** Тонкая серая (rgba(0,0,0,0.05))
- **Тень:** Мягкая черная (rgba(0,0,0,0.08))
- **Border-radius:** 12px

### 6. Login страница
- **Фон:** Градиент от черного к оранжевому
  - `linear-gradient(135deg, #1a1a1a 0%, #FF6B00 100%)`
- **Карточка:** Белая с логотипом в центре
- **Кнопка входа:** Оранжевый градиент

```
╔════════════════════════════════╗
║  🟠 Оранжево-черный градиент  ║
║                                ║
║    ┌──────────────┐            ║
║    │   [LOGO]     │            ║
║    │ Strong Admin │            ║
║    │  [Username]  │            ║
║    │  [Password]  │            ║
║    │   [🟠Войти]  │            ║
║    └──────────────┘            ║
║                                ║
╚════════════════════════════════╝
```

## 💡 Правила использования

### ✅ Правильно

1. **Оранжевый для действий:**
   - Активные состояния
   - Кнопки
   - Акценты
   - Интерактивные элементы

2. **Черный для структуры:**
   - Заголовки
   - Навигация
   - Текст
   - Границы

3. **Белый для контента:**
   - Фон карточек
   - Области контента
   - Формы

### ❌ Неправильно

- ❌ Не используйте оранжевый для больших фоновых областей
- ❌ Не используйте черный текст на оранжевом фоне (плохая читаемость)
- ❌ Не смешивайте другие яркие цвета с оранжевым

## 🔍 Контрастность

| Белый | Оранжевый (#FF6B00) | 4.52:1 | ✅ AA |
| Черный | Белый | 21:1 | ✅ AAA |
| Оранжевый | Белый | 4.52:1 | ✅ AA |
| Белый | Черный | 21:1 | ✅ AAA |

## 🔧 Как изменить ц��ета

### Изменить оттенок оранжевого

Откройте `src/theme/theme.ts`:

```typescript
primary: {
  main: '#FF7700',      // Более яркий оранжевый
  // или
  main: '#FF5500',      // Более темный оранжевый
}
```

### Изменить черный на темно-серый

```typescript
secondary: {
  main: '#2d2d2d',      // Темно-серый вместо черного
}
```

### Добавить третий акцентный цвет

```typescript
const brandColors = {
  // ... существующие цвета
  tertiary: {
    main: '#4CAF50',    // Зеленый для success
  },
};
```
